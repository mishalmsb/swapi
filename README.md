## Getting Started

### Prerequisites

- NODE.JS: ^16.14.0
- NPM: Recommended 8.5.5

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mishalmsb/swapi.git
   ```
2. Install NPM packages
   ```sh
   cd swapi
   ```
3. Install NPM packages
   ```sh
   npm install
   ```

### Run the application

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.
